# Riya's Portfolio

A personal portfolio website built with React, showcasing my skills, projects, and certifications.

## 🔗 Live Demo
[View Portfolio](https://portfolio-rho-blush-19.vercel.app?_vercel_share=Qq6vhRK6mnJ2mWzn0yZOKRLQlziwJSgT)

## 🛠️ Built With
- React
- CSS

## 📌 Sections
- **Home** – Introduction and hero section
- **About** – A little about me
- **Skills** – Technologies I work with
- **Certifications** – Courses & credentials I've earned
- **Projects** – Things I've built
- **Contact** – Get in touch

## 🚀 Run Locally
```bash
git clone https://github.com/Riya-git257/portfolio.git
cd portfolio
npm install
npm start
```
